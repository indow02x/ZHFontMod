# 🗡️ Slay the Spire 2 - 中文字體替換模組 (ZHFontMod)

**作者 (Author):** 亡音星啟  
**版本 (Version):** v1.0.0  

本模組專為追求最佳視覺與閱讀體驗的中文玩家打造！雖然《殺戮尖塔 2》原生的中文字體已經相當完整，但透過本模組，你可以將全遊戲的字體自由替換成更符合個人喜好、更清晰舒適的樣式，讓長時間的卡牌構築之旅更加護眼與享受。

## ✨ 功能特色 (Features)

* **預設高質感字體**：內建大家最熟悉、閱讀體驗極佳的「微軟正黑體」，字型邊緣更平滑清晰，大幅提升介面、卡牌與遺物文本的閱讀舒適度。
* **精準無縫替換**：透過 Harmony 動態攔截原廠字體載入邏輯，安全替換 `zhs` (或自訂) 語言的遊戲字體，不破壞遊戲核心檔案。
* **隨插即用**：採用動態路徑定位 (`System.Reflection`)，無論你的遊戲安裝在哪個磁碟槽，Mod 都能自動尋找並載入同資料夾下的字體檔。
* **自訂專屬風格**：如果你想使用其他字體（例如：思源黑體、jf open 粉圓等），只需將你的字體檔案重新命名為 `zh_font.ttf` 並覆蓋原檔案即可。
* **乾淨輕量**：沒有肥大的依賴庫，不會干擾遊戲核心檔案。

## 📥 玩家安裝教學 (Installation for Players)

如果你只是一般玩家，想要在遊戲內使用這個字體，請按照以下步驟操作：

1. 前往本專案的 [Releases 頁面](這裡貼上你的GitHub Release連結) 下載最新的 `v1.0.0` 壓縮檔。
2. 解壓縮檔案，你應該會看到以下 3 個檔案：
   * `ZHFontMod.dll` (模組核心程式)
   * `zh_font.ttf` (字體檔案 - 微軟正黑體)
   * `metadata.json` (模組資訊檔)
3. 找到你的《殺戮尖塔 2》遊戲安裝目錄，並進入 `Mods` 資料夾 (如果沒有請自行建立)。
   * *通常位於：`C:\Program Files (x86)\Steam\steamapps\common\Slay the Spire 2\Mods\`*
4. 將解壓縮出來的檔案放進 `Mods\ZHFontMod` 資料夾內。
5. 開啟遊戲，在主選單的「模組 (Mods)」介面中啟用本模組，即可享受煥然一新的中文介面！

## 🛠️ 開發者編譯指南 (Build from Source)

如果你是開發者，想要自己修改或編譯這個 Mod，請注意以下事項：

### 環境要求
* .NET SDK (本專案目前設定為 `net9.0`)
* 《殺戮尖塔 2》遊戲本體

### 編譯步驟
1. 複製 (Clone) 本專案到你的電腦上。
2. **⚠️ 重要：** 開啟 `ZHFontMod.csproj`，找到 `<HintPath>` 標籤。
3. 將裡面 `GodotSharp.dll` 與 `sts2.dll` 的絕對路徑，**修改為你電腦上實際的遊戲安裝路徑**。
4. 使用指令 `dotnet build` 進行編譯。
5. 將編譯產出的 `ZHFontMod.dll`、準備好的 `zh_font.ttf` 以及 `metadata.json` 放在同一個目錄下即可測試。
